## 三、核心创新：五层AI Agent逻辑架构（对外答辩高阶架构）
### 架构总述
本产品摒弃传统单层对话式AI、简单链式流水线架构，采用**行业标准高阶五层AI智能体架构**：
**感知层 → 记忆层 → 规划层 → 执行层 → 迭代层**

> 重要说明：
> 五层为**产品逻辑智能层级（对外展示AI原生性、创新性）**
> 工程落地采用**极简整合式开发（保证极低开发难度、极高稳定性）**
> 学术、工程完全自洽，无逻辑漏洞

### 3.1 第一层：感知层（Perception Layer）
**核心能力：全域信息采集与输入**
1. 平台规则感知：实时识别微信公众号、视频号算法规则、合规红线、流量偏好
2. 赛道竞品感知：捕捉垂类爆款结构、选题趋势、内容缺口
3. 用户账号感知：识别用户账号阶段、文风、内容偏好、数据表现

**技术实现**：公开合规数据源 + Supabase知识库预存储 + 用户信息录入

### 3.2 第二层：记忆层（Memory Layer）【项目脑 + 多模态内容库双层记忆】

当前工程实现采用 **Supabase PostgreSQL + Markdown 记忆 + 结构化项目脑** 的双层记忆体系，并已集成 **多模态向量 RAG 能力**：

#### （1）公共赛道内容库（全局共享）+ 多模态向量检索

存储：垂类爆款标题公式、内容结构模板、排版规范、合规词库、平台规则、热点素材库

- **支持内容类型**：公众号文章 + 视频号脚本 + 爆款视频分析
- **向量化模型**：Doubao-embedding-vision-251215（多模态，支持文本 + 视频 URL 分析）
- **检索方式**：关键词检索 + 语义向量检索（混合检索）

#### （2）用户私有项目脑（千人千面）

存储：用户专属文风特征、历史爆款内容、选题偏好、复盘数据、账号成长轨迹、当前阶段、下一步动作

**技术优势**

- 单数据库统一管理，轻量化、免费、高稳定
- 天然支持用户数据隔离、权限安全
- 多模态向量能力，可分析视频 URL 和文本内容
- 当前可见性强，便于在聊天、项目卡、增长分析中持续复用

### 3.3 第三层：规划层（Planning Layer）
**核心能力：任务拆解、智能决策、路径规划**
由中心调度智能体完成：
1. 解析用户创作需求
2. 判断用户账号阶段与创作目标
3. 自主规划任务顺序：诊断→分析→选题→创作→合规→排版→复盘
4. 根据不同垂类、不同账号状态动态调整创作策略

> （工程极简优化：规划逻辑采用固定最优工作流Prompt固化，无需复杂动态调度）

### 3.4 第四层：执行层（Execution Layer）【ToolCalling + Workflow】
规划层下发任务后，执行层通过**标准化 ToolCalling + 关键链路 workflow 化**完成全流程落地：

#### 内置标准化工具池
1. `tool_rag_retrieve`：Supabase向量知识库检索，消除AI幻觉
2. `tool_account_diagnose`：账号定位与阶段诊断
3. `tool_competitor_analysis`：竞品爆款结构拆解
4. `tool_content_create`：文案/脚本智能生成
5. `tool_sensitive_check`：微信生态合规风控检测
6. `tool_style_render`：公众号/视频号智能排版渲染
7. `tool_data_review`：内容数据归因复盘

**AI原生核心证明**
不是简单文本生成，而是：

- 大模型自主判断
- 工具调用
- 外部知识检索
- 结构化输出
- 关键链路 workflow 化

当前“增长分析”已经作为第一条 LangGraph workflow 试点落地。

### 3.5 第五层：迭代层（Iteration Layer）
**核心能力：自我进化、数据闭环、持续优化**
1. 采集用户发布后数据
2. 复盘内容优劣归因
3. 自动更新用户私有记忆库
4. 下一轮创作自动适配优化策略
实现**越用越懂用户、越用越精准**的AI自进化闭环。
